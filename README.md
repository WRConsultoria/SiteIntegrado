# Planos de Consultoria

Este diretório contém a página de apresentação dos planos de consultoria (Básico, Completo e Premium).

## Estrutura

- `index.html`: Página principal que lista os planos, preços e links para mais informações.
- `assets/`: Contém recursos estáticos como imagens, estilos e PDFs com detalhes dos planos.

## Funcionalidades

- Visualização comparativa dos planos.
- Links para download/visualização de PDFs informativos.
- Integração com Mercado Pago para pagamentos (links diretos).
